# Hotel-Room-Booking-Web-App
A Hotel Room Booking Web Application developed using Spring Boot, Thymeleaf, MySQL, JPA, Hibernate, HTML, CSS, Bootstrap, and JavaScript. Users can register, log in, view available rooms, and book rooms, while administrators can manage room details.
# 🏨 Hotel Room Booking Web App

A full-stack Hotel Room Booking Web Application built using Spring Boot, Thymeleaf, MySQL, JPA, Hibernate, HTML, CSS, Bootstrap, and JavaScript.

## Features

- User Registration
- User Login
- Admin Login
- View Available Rooms
- Search Rooms by Type
- Book Rooms
- Room Availability Management
- Responsive UI
- MySQL Database Integration

## Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- Thymeleaf
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Maven
